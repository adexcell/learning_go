# Распаковка строки

Реализована функция Go, осуществляющая примитивную распаковку строки, содержащей повторяющиеся символы/руны.

**Примеры работы функции:**

```
Вход: "a4bc2d5e"
Выход: "aaaabccddddde"

Вход: "abcd"
Выход: "abcd" (нет цифр — ничего не меняется)

Вход: "45"
Выход: "" (некорректная строка, т.к. в строке только цифры — функция возвращает ошибку)

Вход: ""
Выход: "" (пустая строка -> пустая строка)

Вход: "qwe\4\5"
Выход: "qwe45" (4 и 5 не трактуются как числа, т.к. экранированы)

Вход: "qwe\45"
Выход: "qwe44444" (\4 экранирует 4, поэтому распаковывается только 5)

Вход: "a10b2"
Выход: "aaaaaaaaaabb"

Вход: "a10b2"
Выход: "aaaaaaaaaabb" (корректная работа с многозначными числами)

Вход: "ф3в4"
Выход: "фффвввв" (корректная работа с рунами)

Вход: "a^$b2"
Выход: "" (некорректная строка, т.к. функция обрабатывает только буквы, цифры и \ — функция возвращает ошибку)
```

**Запуск программы с заготовленными примерами:**
`make run`

**Запуск тестов:**
`make test`

**Анализ кода с vet и golint:**
`make check`
